# VREED Eye Tracking Analysis

Exploratory analysis and visualization of eye tracking data from the VR Emotion Recognition Dataset (VREED).

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Place the VREED dataset under `dataset/` (or symlink it there).

## Dataset

Tabbaa, L., Searle, R., Bafti, S. M., Hossain, M. M., Intarasisrisawat, J., Glancy, M., & Ang, C. S. (2022). VREED: Virtual Reality Emotion Recognition Dataset Using Eye Tracking & Physiological Measures. *Proceedings of the ACM on Interactive, Mobile, Wearable and Ubiquitous Technologies*, 5(4). https://doi.org/10.1145/3495002

Available on [Kaggle](https://www.kaggle.com/datasets/lumaatabbaa/vr-eyes-emotions-dataset-vreed).
